Part 01
核心交易哲学

"逻辑派"身份定位

Coming 反复强调自己是"逻辑派"而非"技术派"。他的交易不以止损线和技术形态为核心依据，而是围绕估值、情绪、结构和数据做出判断。核心信条：

数据推导逻辑，逻辑反映到涨跌；逻辑不会在一天之内兑现，但给足时间，逻辑一定会兑现。

—— 2026.02.06

他认为"投资投的是数据分析能力、遵守交易纪律、逻辑与认知"，而大部分人投的只是涨跌，所以亏钱。

"实战派"立场

从星球创立初期（2025年7月）起就明确声明：自己分享的是长期持有100手、过亿名义市值的实战经验，而不是纸上谈兵。所有逻辑都对应着真金白银的仓位操作。

择时核心——估值与情绪的双锚

贯穿全部九个月的最根本择时原则：

🧊
哀鸿遍野时开仓
情绪极端悲观 → 做多。不追求精准抄底，而是在恐慌中分批布局。
🔥
行情沸腾时不适合作为起点
情绪极端乐观 → 减仓或做空。建立在"估值位置 + 情绪状态"双重确认之上。

"偏见"与"定力"的区分

2026年2月提出的精炼判断标准：如果一个逻辑每天都被事实削弱，那叫偏见；如果逻辑每天都被事实加强，只是价格还没反应，那叫定力。认错的条件不是价格创新高，而是核心逻辑变量是否被证伪（例如成交量是否破前高）。

Part 02
核心交易逻辑月度演变
2025年7月
体系起点——吃贴水策略框架确立
核心理念：吃贴水不是短线暴富工具，起点选择很重要，估值与情绪是择时核心。分层布局（多个点位分散下注），不把筹码集中在单一点位。
2025年8月
顶部风险渐成——平滑减仓 + 期权防御
判断牛市未结束但阶段性顶部风险越来越大。策略转向"平滑减仓 + 买put/卖call + 留足子弹"。不主张普通人做空，核心态度是"进退有余"。月末确立"微盘股1700"为阶段性决策锚点。
2025年9月
坚定看空微盘股——锚定1700
全月锚定"微盘股1700"。方法论：贴水突然变大往往预示大波动、微盘股是最关键先行指标。执行上不激进梭哈，持续平IF多单、持有IM空单与put、动态减仓与换月。
2025年10月
负反馈滚雪球——模型化回调框架
完整提出"位置高→情绪脆弱→小票吸血→成交量萎缩→负反馈滚雪球式下跌"。形成14%均值回撤、20%回撤概率60%、情绪冰点6261、进三退一筹码分布位等模型化回调框架。
2025年11月
看空逻辑与做空成本分离
核心矛盾：逻辑正确 ≠ 做空值得持有。月中一度"认输投降空仓观望"，月下旬微盘股转弱后重新启动做空。核心教训：贴水、逼仓、资金占用、时间成本都很高，空仓等触发条件比硬扛更优。
2025年12月
调整成局——跨品种交易展开
核心逻辑链：微盘股先跌→IM跟跌→情绪杀→大调整。月末部分切换到黄金/白银VIX交易，展现跨品种迁移能力。同时暴露仓位控制问题。
2026年1月
套利、逼仓识别与高压实战
IM/IC价差套利成为核心策略。经历两种逼仓（普涨式 vs 定向攻击500/1000）。仓位管理成为生死问题。核心认知：逼仓是一种可被辨识的市场结构，不是不可预见的随机事件。
2026年2月
健康调整预期成型——方法论成熟
从"多300空500/1000"过渡到"健康调整"高置信度判断。威科夫结构、人工数据整理+AI辅助、多指标交叉验证。"逻辑已收束，只差路径选择"。
2026年3月
ABCDE结构推进——石油危机叠加
围绕ABCDE点位结构交易。C点附近完成28手减仓，策略从进攻切换到等待。石油危机（伊朗封锁霍尔木兹海峡）成为宏观驱动核心。原油→制造业利润→中证1000盈利脆弱性链条清晰化。
Part 03
交易方法论体系

择时框架——"位置轴 + 时间轴"

2026年3月正式提出双轴框架。位置轴回答"在哪里交易"（估值高低、关键点位、价差极值），时间轴回答"什么时候交易"（情绪到位、资金枯竭、节后行为金融特征等）。两轴配合才构成完整的交易决策。

先行指标逻辑——"头与身体"

微盘股是行情的龙头和情绪急先锋，沪深300是身体。如果龙头涨不动、主力也涨不动，就说明是强弩之末。

—— 2025.09.12

调整时微盘股先行下跌，再映射到中证1000和IM。历史涨幅比例约3:1，下跌也按类似比例扩散。

仓位管理哲学

⚖️
仓位舒服 > 观点正确
看对方向不等于可以无限加仓。"80手太多了，回到心理能接受的50手。"
📉
平滑减仓
不一次清仓，分批有序退出。每次减仓量根据情绪和位置动态调整。
💰
保留子弹优先级最高
下跌后有足够子弹开多单，比赌方向更重要。"为了保留最充足的子弹。"
📋
用交易计划代替猜测
预设不同情境的操作方案，如"反弹到8050加12手，到8180恢复48手"。

期权运用框架

卖 Call
基于估值观点和情绪主导趋势，在高VIX环境下性价比极高。
买 Put
失败亏损有限、成功数倍利润，同时不会增加太多风险敞口。顶部防御系统的核心组件。
末日期权彩票
对冲"最后一涨"风险。买入成本极低的末日期权，以极小代价覆盖尾部风险。
跨期套利
恐慌杀跌时做"空近月+多远月"，赚取远月过度恐慌修复的收益，而非方向上涨的钱。

吃贴水策略的适用边界

Coming 明确承认：吃贴水策略更适合行情上升段，熊市里大部分时候并不适用。做空要烧掉每月100多万贴水，位置高不等于立刻跌，所以有时不如空仓等一个绝对情绪高点再做空，或者干脆放弃做空。看空逻辑正确 ≠ 做空交易划算——这是11月最核心的教训。

风控铁律

杠杆低一点。

—— 被问到风控铁律时的极简回答，2025.10.25

围绕这一核心，延伸出：保证金管理、风险敞口控制、时间止损（"空单再持有两周，不盈利就投降"）、不追求零回撤、不因看对方向就无限加仓。

Part 04
监控指标分类与逻辑说明

一、情绪类指标

指标 分类 逻辑说明
PCR
Put-Call Ratio		 情绪 最高频使用的情绪指标。PCR处于极值时判断市场恐慌/贪婪程度。但强调PCR只是多指标中的一个，不能单独使用。PCR见顶后的前三根阴线有统计规律，可作为"盘中剧本"参考。PCR落到中值以下则存在超跌反弹需求。
情绪指数
韭圈 / 芝士财富		 情绪 第三方情绪指数。情绪指数74属于偏低尚未冲顶，沸腾时对应极高值。情绪到"冰点"是底部区域的必要条件之一。与PCR、成交量等交叉验证使用。
VIX
波动率指数		 情绪 VIX接近阶段极值 = 历史级别波动。高VIX下卖call性价比极高。VIX是跨品种共振（黄金/白银/股指）的桥梁。VIX爆拉意味着"阶段性高点中的高点、鱼尾中的鱼尾"。"历史级别VIX"对应做对方向时可以吃到大段利润。
群体情绪定性
社群观察 / 抄底人数		 情绪 包括星球群情绪观察、抄底人数统计。"真正的底部不可能被那么多人抄到"——大量人抄底时绝不可能是底部。"人声鼎沸时等待，鬼哭狼嚎时入场"。财经自媒体开始转载逻辑时说明信息差正在消失。

二、资金与量能类指标

指标 分类 逻辑说明
成交量 量能 最核心的量能观测变量。"没有成交量的上涨最终都是诱多"——缩量上涨 = 脆弱上涨。成交量萎缩（如从4万亿到2.9万亿再到2.5万亿以下）是下跌确认的重要证据。成交量破前高才是空头需要真正认错的时候，而非价格创新高。下跌开始后成交量三天内会快速萎缩。
融资余额
融资交易比例		 量能 融资余额进入下降通道但尚未到杀杠杆阶段时，说明市场退潮但还没到最惨烈阶段（如26200亿时）。融资余额未上升 + 小票多头看似强 → 实则空头占优势，一旦没有后续资金接力泡沫会一溃千里。
ETF净申购/赎回 量能 ETF净申购约150亿 → 短期超跌反弹条件。ETF赎回 → 判断机构/国家队行为的线索。卫星ETF（小票类ETF）的表现被视为散户资金方向的代理变量。
主力资金流向 量能 辅助验证。主力资金5天流出8000亿时佐证"这里不应该涨"的判断。国家队主要卖300时消耗了300的流动性并压制股价。

三、价差与基差类指标

指标 分类 逻辑说明
IC/IM 贴水（基差） 价差 最核心的结构指标。贴水突然变大 → 通常意味着短期有较大波动，更倾向于大跌。基差没有走扩在历史中是非常诡异的情况，需要特别警惕（可能是量化吃掉了套利超额）。期货基差不转为贴水 → 后续还会有逼仓。IC/IM 贴水38点 + 2.2万亿成交符合看空逻辑。
500/1000 价差 价差 价差接近阶段顶点（如590接近600）→ 减仓信号。价差过小 → 市场情绪没那么热，不合理定价没那么严重，也可能意味着1000跌得没那么多。所有不合理的价差最终都会回归。
IM/IC 价差 价差 价差非常接近0时 → 往往对应相对底部与后续一波上涨。是确定性较高的套利机会。"多IM空IC，3个月内几乎百分百可以套利2万一手"。
近远月价差 价差 近远月价差缩小 → 可能意味着"要涨了"。恐慌时远月流动性差更容易被砸出极端深贴水 → 可做"空近月+多远月"的跨期套利。恐慌退潮后远月修复收益显著。
升贴水结构 价差 升水没有大幅提升 → 空头有利信号之一。天量升水出现后修正的过程是派发结构的关键识别点。1.23修正后出现天量升水 → 逼仓周期结束的确认信号之一。

四、市场结构类指标

指标 分类 逻辑说明
微盘股指数 结构 全体系中最核心的先行指标。"微盘股跌了，IM肯定崩。"微盘股与中证1000按约3:1的比例映射涨跌幅。微盘股崩盘 = 重新做空的触发器。微盘股到1700 = 阶段性决策锚点（平部分空仓、判断泡沫释放程度、决定何时重新吃贴水）。
涨跌家数 结构 全市场下跌家数高是佐证下行结构的辅助变量。"场内只有散户，没有历史盈利资金或决策资金"时更有意义。
换手率 / 天量换手 结构 天量换手后的行为模式是判断顶部派发的关键。如1.14天量换手 → 后续12.17~2.12逼仓周期 → 派发结构确认。换手率与成交量配合判断节奏。
逼仓识别 结构 区分两种类型：普涨式逼仓（全市场普涨，如1.14）和定向逼仓（重点攻击500和1000，如1.23）。两者对后续结构影响不同。逼仓是一种可被辨识的市场结构，不是不可预见的随机事件。
平均股价 / 散户心理位 结构 "平均股价再跌1%就到散户心理恐慌位"——散户心理恐慌位的接近程度是判断恐慌盘是否即将出现的辅助依据。

五、宏观与外部变量

指标 分类 逻辑说明
原油 / 石油危机 宏观 2026年3月的核心宏观驱动。传导链条：伊朗封锁霍尔木兹海峡 → 原油高起 → 中证1000净利润率仅3.86%（vs 300的12%），对油价冲击更脆弱。原油不只是汽油，而是工业生产源头。中国是进口国受损更大。
美联储利率 / 降息预期 宏观 不降息已成定局 → 利率预期重估。加息概率抬升被纳入市场风险定价。利率是"真金白银"，与石油危机形成双重压力。降息属于利好出尽。9.18降息后预期进入"牛2"阶段。
港股走势 宏观 港股先行下跌是A股调整的先行信号。IM与日经、纳指、金龙等全球风险资产走弱具有联动性。港股先行回落 → 微盘股上攻乏力 → A股调整概率上升。
国债走势 宏观 国债涨 + 300涨 + 50涨 + 成交量萎缩 + 小票被吸血 = 风格极度分化的结构信号。国债不跌时猜测是快到顶了、有防守资金进去。
停火概率 宏观 地缘冲突的停火概率结构变化被纳入盘前因子清单。短期略有改善不代表风险解除。"船过不去"是短期内无法改变的硬约束。
人民币汇率 宏观 被视为"一根稻草"级别的边际变量，不影响主逻辑。调整外汇准备金率本质是减震器，最多影响短期情绪，不是趋势改变者。

六、期权专用指标

指标 分类 逻辑说明
MO 隐波
隐含波动率		 期权 MO隐波到38附近 ≈ 500/1000的短期低点。隐波的位置决定了期权策略的性价比——高隐波利于卖方（卖call），低隐波利于买方（买put）。
期权时间价值消耗 期权 持有650张put、价值380万时每天消耗时间价值25万。期权做得好不好决定最终战绩。期权最肥美的时候过去后需要主动减仓锁定利润。时间拖太久期权会归零。

Part 06
交易纪律与心理管理精要
不为交易而交易
"不做"本身就是最高期望值的选择。投资不是比勤奋的游戏，近几个月没怎么动不代表不在工作。
5%人盈利、95%人亏损
不能跟着大多数人想法走。期货市场的本质决定了必须站在多数人的对立面思考。
趋势市里指标钝化时的应对
把交易也钝化——逐步推进，不要一下满仓一下清仓。用渐进节奏匹配指标失灵期。
拍脑袋想，不如跟着数据走
当数据与主观感受冲突时，以数据为准。缩量不能孤立解读，要和多个数据一起对照看。
预案管理
预设极端情况（如IM/IC单日跌6%）的应对方案。冷静预案优于情绪上头。"放弃幻想，早做准备。"
不耍小聪明
硬吃回撤，不在C→D段做短线交易来"优化"净值曲线。"就像年后硬吃红包行情的回撤一样，这是投资过程之不可避免的部分。"
对国家队保持敬畏
"曾经与国家队作对所以亏钱。"国家队不会和散户"过家家"做短差。散户以为自己比国家队聪明是危险的幻觉。
投资不缺机会，耐心等待
"在人声鼎沸时等待。"真正高胜率的机会需要情绪极端才会出现，中间的大部分时间应该什么都不做。

如果没有一手消息，就不要对消息下注。逻辑是为估值服务的。不要怀疑，只需要等合理的事发生。行情最高点是被最疯狂的人推上去的，不需要理解疯子。
